## What is ValidationForge?

ValidationForge is a no-mock validation platform for Claude Code. It replaces guesswork with real evidence — actual builds, actual interactions, actual proof your code works.

## Key Features

<CardGrid stagger>
  <Card title="No Mocks Ever" icon="warning">
    Never create test files, mocks, stubs, or test doubles. Validate against the real system.
  </Card>
  <Card title="Evidence-Based" icon="document">
    Every PASS/FAIL cites specific evidence: screenshots, logs, API responses.
  </Card>
  <Card title="7-Phase Pipeline" icon="list-format">
    Research → Plan → Preflight → Execute → Analyze → Verdict → Ship
  </Card>
  <Card title="Multi-Platform" icon="laptop">
    iOS, Web, API, CLI, Fullstack — validate any platform with the right validator.
  </Card>
